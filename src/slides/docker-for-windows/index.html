Solution? Virtualization!

  • Difficult maintain and share
  • Time consuming to setup
Vagrant
  • Still bulky
  • Easier to maintain and share

Docker to the Rescue!

Docker
  • Tiny footprint
  • No need to share local instance
  • "Two-button" setup
