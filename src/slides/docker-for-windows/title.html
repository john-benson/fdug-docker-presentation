Development in Windows can be a nightmare
  • Non-native implementations (Redis)
  • Difficult to get libraries (NumPi)
  • Missing features, bugs, quirks
  • Completely unlike your production environment

Solution? VMs!

Virtual Machines
  • Bulky
  • Difficult maintain and share
  • Time consuming to setup
  • Comparatively slow
Vagrant
  • Still bulky
  • Easier to maintain and share
  • Still time consuming to setup
  • Still comparatively slow

Docker to the Rescue!

  • Native HyperV implementation
  • No image mangement. Docker takes care of everything
  • The same "bits" that run on Linux
  • Easy to share cross-OS
